# Object-Oriented Programming (OOP) – Route .NET Backend Diploma

This folder contains my assignments, exercises, and practice work from the **Object-Oriented Programming (OOP) course** in the Route .NET Backend Diploma.

The course focuses on mastering **object-oriented programming principles in C#**, including classes, inheritance, polymorphism, encapsulation, abstraction, and designing maintainable software systems.

---

## Topics Covered

### Session 01 – OOP Fundamentals

* OOP definition and why it matters
* User-defined data types
* Classes and object creation
* Constructor overloading and constructor chaining
* Structs and lightweight value types
* Class vs Struct comparison
* Class libraries and code organization
* Access modifiers (`private`, `public`, `internal`)

### Session 02 – Encapsulation & Class Features

* Encapsulation concept and implementation in C#
* Properties in C#
  * Types of properties
  * Field vs property
  * Best practices
* Indexers in C#
  * Syntax and real-world usage
  * Indexer overloading
* `static` keyword
  * Static fields, methods, constructors, and classes
  * Instance vs static members

### Session 03 – Class Relationships & Inheritance

* Class relationships
  * Inheritance (IS-A)
  * Association
  * Aggregation & Composition (HAS-A)
  * Dependency
* Inheritance in C#
  * Syntax and constructors
  * Limitations and Liskov Substitution Principle (LSP)
  * Composition vs inheritance
* Access modifiers with inheritance
* Sealed classes and methods
  * `sealed` class
  * `sealed` method
  * When and why to use them

### Session 04 – Polymorphism

* Polymorphism in C#
  * Definition and importance
  * Static vs dynamic types
* Polymorphism techniques
  * Method overloading
  * Operator overloading
  * Method hiding
  * Method overriding
* Binding concepts
  * Static binding
  * Dynamic binding
* Practice and exercises
  * Code output questions
  * Discount system example
  * Payment system practice

### Session 05 – Interfaces & Object Copying

* Interfaces in C#
  * Simple interfaces
  * Using interfaces with polymorphism
  * Implementing multiple interfaces
  * Interfaces as parameters
* Advanced interface features
  * Explicit interface implementation
  * Default interface methods (C# 8+)
* Object copying
  * Reference copy
  * Shallow copy
  * Deep copy
  * `ICloneable` interface
  * Immutability

### Session 06 – Abstraction & Advanced Features

* Abstraction in OOP
  * Core concept and real-world examples
  * Good vs bad design
  * Common mistakes
* Abstract class vs interface
  * Decision rules
  * Partial implementation
* Partial classes
  * Syntax and compilation
  * Real-world usage
* Partial methods
* Extension methods
  * Defining and using extension methods
  * When to use and when to avoid them
* Mini exercises and challenges

# Advanced C# – Route .NET Backend Diploma  

This folder contains my assignments, exercises, and practice work from the **Advanced C# course** in the Route .NET Backend Diploma.

The course focuses on mastering **advanced C# concepts**, including generics, delegates, collections, and asynchronous programming, to build efficient, scalable, and high-performance applications.

# Database Course – Route .NET Backend Diploma

This folder contains my assignments and practice exercises from the **Database course** in the Route .NET Backend Diploma.

The course focuses on relational database design and SQL Server development, covering everything from **ERD modeling** to **advanced database objects and security concepts**.

---

## Topics Covered

### Session 01 – Database Design & ERD

* Introduction to databases and why ERDs matter
* Entities and attributes
* Keys and identifiers
* Relationships and cardinality
* Advanced concepts (specialization and weak entities)
* Best practices in ERD design

### Session 02 – Relational Database Concepts

* Relational database definitions
* Mapping rules from ERD to relational schema
* Practical exercises

### Session 03 – SQL Server Basics

* Introduction to SQL Server
* Comments and variables
* SQL Server data types
* Database schema
* Data Definition Language (DDL)

### Session 04 – Data Manipulation

* DDL using SQL Server wizard
* Data Manipulation Language (DML)
* Referential integrity rules

### Session 05 – Querying Data

* Referential integrity rules
* Data Query Language (DQL)
* SELECT statement
* Joining tables

### Session 06 – Advanced Querying

* TOP clause
* OFFSET & FETCH
* Ranking functions
* SQL query execution order

### Session 07 – SQL Functions & Aggregation

* Built-in SQL Server functions
* GROUP BY clause
* Subqueries

### Session 08 – Advanced SQL Features

* Set operators (UNION family)
* Common Table Expressions (CTE)
* SELECT INTO
* User-defined functions (UDF)

### Session 09 – Indexing

* Introduction to indexes
* Index performance concepts
* Practical exercises

### Session 10 – Views

* Creating and managing views
* Comparison between Views, CTEs, and Temporary Tables

### Session 11 – Stored Procedures & Triggers

* Stored procedures
* Parameters and execution
* Error handling with TRY/CATCH
* Trigger types and use cases
* INSERTED and DELETED tables
* Security and best practices

### Session 12 – Transactions & Triggers

* Trigger best practices
* Transaction Control Language (TCL)
* BEGIN, COMMIT, and ROLLBACK
* SAVEPOINT and nested transactions
* Error handling in transactions

### Session 13 – Security & Database Control

* LOGON triggers
* Transaction handling best practices
* Data Control Language (DCL)
* Login vs User
* Server roles and database roles
* GRANT, REVOKE, and DENY

# C# Basics – Route .NET Backend Diploma

This folder contains my assignments, exercises, and practice notes from the **C# Basics course** in the Route .NET Backend Diploma.

The course focuses on learning **C# programming fundamentals** and **.NET development**, covering everything from **basic syntax and data types** to **object-oriented programming principles and memory management**.

* Key tools and setup for development

* Comments in C#
